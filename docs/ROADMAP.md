# Roadmap

## Round 1 (this release)

Foundation: registry, tasks, lifecycle, routing, budgets, context, git inspect, manual adapters, reports, behavioral metrics, CLI, docs, tests.

## Round 2 (candidates)

- Richer plan artifacts and approval gates in the CLI
- Stronger path sandboxing against real filesystem diffs
- Optional local metrics dashboards (still offline)
- Better resume/idempotency for long tasks

## Round 3+ (only with explicit approval)

- Equitify connection after the exact user command
- Any future automation beyond manual handoff
- Optional integrations that still avoid unpaid/unapproved network model spend

## Explicitly deferred

- LangChain / CrewAI / AutoGen
- Paid API adapters
- Browser automation
- Auto-merge / auto-push
- Self-modifying routing rules
